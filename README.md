# React Quiz App - [Live](https://isdestiny.github.io/react-quiz-app)

Quiz app that utilizes the [open trivia db](https://opentdb.com/). Questions can either be true/false or 
multiple choice. 

## Quiz Categories
- General Knowledge
- Computers
- Anime
- Sports
- History
- Politics

## Tech Stack:
- Javascript
- React
- Styled Components
- React Context/Reducer
